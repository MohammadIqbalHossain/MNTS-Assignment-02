# How to run the project locally.

Step-01:

open a teminal, run this command

```commad
cd youDirectory
```

Clone full code from the GitHub in your machine.

```command
npm clone https://github.com/MohammadIqbalHossain/MNTS-Assignment-02.git
```

Install all depdencies for the project.

```command
npm i
```

You'll find a script in `package.json` file that runs the project while development phase before compiling the file in js.

```js
npm run star:dev
```

You'll find another script to compile the project.

```js
npm run build
```
